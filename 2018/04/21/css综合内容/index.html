1.编码规范

平时写代码所遵循的规范如下：

  1. 命名都用小写，多个单词时用中划线连接。
  2. 命名用有意义的单词，做到看到名字就明白是干什么的。
  3. 多条属性时，每条属性都占一行，不写在一行。
  4. 每条声明语句的 : 后应该插入一个空格。
  5. 对于以逗号分隔的属性值，每个逗号后面都应该插入一个空格box-shadow: 0 1px 2px #ccc, inset 0 1px 0 #fff

尽可能把 http://codeguide.bootcss.com/ 都做到
